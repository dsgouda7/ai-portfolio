## Key Pedagogical Elements

### What Makes the MI Treatment Complete:

1. **Concrete numeric example** (8 districts, 2×2 contingency table)
   - Shows p(x,y), p(x), p(y) computation
   - Log ratio calculation with actual numbers
   - Final MI = 0.693 bits interpretation

2. **Entropy as uncertainty** explained with:
   - Binary outcome entropy values (0, 0.72, 1.0 bits)
   - H(Y) - H(Y|X) formula grounded
   - Visual bar chart showing reduction

3. **Why Pearson fails** for each case:
   - U-shape: product cancellation shown explicitly
   - Threshold: averaging dilutes the cliff
   - Parabola: numeric table showing (-18, -2, +2, +18) cancellation

4. **Practical computation**:
   - sklearn's k-NN estimator explained
   - Parameters (n_neighbors, random_state)
   - When it fails (small n, high dim)

5. **Decision framework**:
   - When Pearson and MI agree → linear or absent
   - When MI >> ρ² → non-linear signal found
   - Which to use for which model type
