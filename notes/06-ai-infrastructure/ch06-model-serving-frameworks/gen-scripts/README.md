# Ch.6 Model Serving Frameworks — Diagram Generation Scripts

This directory contains Python scripts to generate all diagrams and animations for Chapter 6.

## Prerequisites

```bash
pip install matplotlib numpy pillow
```

## Generate All Diagrams

Run all scripts to generate the complete set of visualizations:

```bash
# Throughput comparison bar chart
python gen_ch06_throughput_comparison.py

# KV cache memory usage line chart
python gen_ch06_kv_cache_usage.py

# Serving architecture diagram
python gen_ch06_serving_architecture.py

# Framework decision tree
python gen_ch06_decision_tree.py

# Animated throughput needle (GIF)
python gen_ch06_serving_throughput_needle.py
```

All outputs are saved to `../img/`.

## Generated Files

- `ch06-throughput-comparison.png` — Bar chart: HuggingFace (10 req/s) vs ONNX (24 req/s) vs vLLM (147 req/s) vs TensorRT (189 req/s)
- `ch06-kv-cache-usage.png` — Line chart: Memory scaling with batch size for different frameworks
- `ch06-serving-architecture.png` — Production deployment: Load balancer → 3× vLLM replicas → Prometheus monitoring
- `ch06-decision-tree.png` — Framework selection flowchart (LLM? Throughput? GPU vendor? Latency?)
- `ch06-serving-throughput-needle.gif` — Animated speedometer: 10 req/s → 150 req/s transition

## Design Conventions

All diagrams follow the AI Infrastructure track visual style:

- **Dark background:** `#1a1a2e` (figure), `#16213e` (axes)
- **Color palette:**
  - Red `#b91c1c`: Baseline/slow (HuggingFace)
  - Orange `#b45309`: Medium performance (TensorRT)
  - Green `#15803d`: Recommended (vLLM)
  - Blue `#1d4ed8`: Monitoring/infrastructure
- **Font:** Default matplotlib (system sans-serif)
- **Resolution:** 150 DPI for all PNG exports
- **Animation:** 100ms per frame, 10 FPS

## Troubleshooting

**Import errors:**
```bash
pip install matplotlib numpy pillow
```

**Font rendering issues on Linux:**
```bash
sudo apt-get install fonts-dejavu-core
```

**Pillow not found (for GIF animation):**
```bash
pip install pillow
```

**Slow GIF generation:**
The animation script generates ~65 frames. This takes 10-15 seconds on most machines. This is expected.
